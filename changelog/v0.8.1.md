# v0.8.1 - Bug Fix

Release Date: 2026-01-15

## Bug Fixes

### URL Trailing Slash Handling
- Fixed repository URL normalization when URL ends with `/`
- Previously: `https://github.com/user/repo/` → `https://github.com/user/repo/.git` ❌
- Now: `https://github.com/user/repo/` → `https://github.com/user/repo.git` ✅

## Files Changed

- `internal/git/fetcher.go` - Trim trailing slashes before adding `.git` suffix
