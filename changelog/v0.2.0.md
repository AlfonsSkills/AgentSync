## üéâ New Features

### One-Click Installation
- **curl-based installation script** (`install.sh`)
  - Auto-detects platform (OS and architecture)
  - Downloads appropriate binary from GitHub releases
  - Installs to `/usr/local/bin`
  - Example: `curl -fsSL https://raw.githubusercontent.com/AlfonsSkills/SkillSync/main/install.sh | bash`

### Interactive Skill Selection
- **Multi-selection interface** for monorepo installations
  - Uses `survey/v2` for interactive prompts
  - Displays skill descriptions from `SKILL.md`
  - Supports keyboard navigation and filtering
  - Always shows selection menu, even for single skills

### Monorepo Support
- **Automatic skill discovery** in repositories
  - Scans for all subdirectories containing `SKILL.md`
  - Supports multiple skills in a single repository
  - Removed direct subdirectory installation (unified under interactive selection)

### Proxy Compatibility
- **Switched to system git command** from `go-git` library
  - Inherits system proxy configurations
  - Resolves network timeout issues
  - Better Git compatibility
