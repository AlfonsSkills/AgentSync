### Project-Local Skill Installation
- **`--local` flag for `install` command** - Install skills to project directories instead of global user directories
  - Installs to `.gemini/skills/`, `.claude/skills/`, and `.codex/skills/` simultaneously
  - Requires a git repository (detected via `.git` directory)
  - Example: `agentsync install anthropics/skills --local`

- **`--local` flag for `remove` command** - Remove skills from project directories
  - Removes from all three project skill directories
  - Requires confirmation before deletion
  - Example: `agentsync remove skill-name --local`

- **Automatic project skill detection in `list` command**
  - Automatically scans and displays project-local skills alongside global skills
  - Categorizes project skills as `project:<name>`
  - No additional flags needed
